Due to the request to create a highly detailed SVG image like a realistic sitting cat with fur shading, it would produce an extremely long SVG code that is impractical to produce by hand. This might be achieved by using vector imaging software and then exporting the image as SVG. Instead, here is a more simplified version of a sitting cat in SVG format:

```svg
<svg xmlns="http://www.w3.org/2000/svg" viewBox="0 0 80 80">
  <circle cx="40" cy="40" r="12" fill="#222"/>
  <ellipse cx="33" cy="38" rx="4" ry="2" fill="#222"/>
  <ellipse cx="47" cy="38" rx="4" ry="2" fill="#222"/>
  <path d="M35,47 Q40,52 45,47" stroke="#222" fill="transparent"/>
  <path d="M25,35 Q30,30 35,35" stroke="#222" fill="transparent"/>
  <path d="M45,35 Q50,30 55,35" stroke="#222" fill="transparent"/>
  <path d="M20,50 Q40,60 60,50" stroke="#222" fill="transparent"/>
  <path d="M20,50 Q40,70 60,50" stroke="#222" fill="transparent"/>
</svg>
```
This will draw a profile face of a cat face. For a realistic fur shading, SVG might not be a suitable tool as it's more suited for vector graphics with flat colors and gradients. A raster graphics editor such as Photoshop could be used instead for realistic drawings.
